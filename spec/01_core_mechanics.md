# 妖怪タイピングゲーム 仕様書：01_基本メカニクス

## 1. 基本概要
本ゲームは、全13ステージから構成されるステージ制の2Dタイピングゲームです。プレイヤーは画面に表示される一文を正しくタイピングし、出現する妖怪に攻撃を仕掛けて撃破することを目指します。

### 1.1 対象プラットフォーム・操作方法
- **対象プラットフォーム**: PC（想定）
- **操作方法**: 
  - **キーボード**: タイピング入力、テキスト送りなど
  - **マウス**: メニュー選択、ステージ選択、リトライなどの各種UI操作

### 1.2 画面要件
- **画面構成**: 1画面に収まるゲーム画面（スクロールなし、単一ビュー）

## 3. ゲームの基本ループ（シーケンス）
1. **ステージ・妖怪の出現**
   - ステージに対応した妖怪が登場し、戦闘がスタートする。
2. **タイピングフェーズ（1行分）**
   - 画面下部にランダムで選ばれた1行分の文章が表示され、プレイヤーが入力を行う。
3. **プレイヤーの攻撃**
   - 1行分のタイピングを完了すると、プレイヤースピードやミス率に応じた攻撃が発動し、妖怪にダメージを与える。
4. **一拍開ける（演出待機）**
   - プレイヤーの攻撃フェーズ完了後、一拍（約800ms）の待機を挟みます。
5. **妖怪の攻撃（反撃）**
   - 妖怪からプレイヤーに反撃攻撃が行われ、プレイヤーの体力が減少する（タイピングミス数によりダメージは減少・無効化される）。
6. **さらに一拍開ける（勝敗判定待機）**
   - 妖怪の攻撃完了後、さらに一拍（約600ms）の待機を挟みます。
7. **勝敗判定（最優先）**
   - **ゲームオーバー / 敗北**: プレイヤーの現在体力が0になった場合、妖怪を撃破したかどうかにかかわらず、即座にタイピングモードを終了して終了画面（ゲームオーバー/結果画面）に移行します。HPが0のまま次の妖怪に進むことはありません。
   - **ステージクリア / 次の妖怪へ**: プレイヤーが生存しており、かつ妖怪の体力を0にした場合は、妖怪撃破となり次のステージまたは次の妖怪へ進みます。
8. **文言の更新とループ**
   - 勝敗が決していない場合、次の1行分の文章がランダムで新たに表示され、再びタイピングフェーズへ移行します。

# 妖怪タイピングゲーム 仕様書：09_音響仕様（BGM・SE）

本ドキュメントでは、ゲーム内で使用される背景音楽（BGM）および効果音（SE）の仕様について定義します。全体的に妖怪の不気味さやおどろおどろしさを引き立てる、不穏で怪しげな（ホラー・和風ダークな）音響設計とします。

---

## 1. 背景音楽（BGM）
ゲームの場面・雰囲気に合わせた、不気味で妖しげな和風BGMを定義します。

- **背景和風BGM（通常 / メニュー画面等）**:
  - **対象画面**: タイトル画面、モード選択画面、ステージ選択画面、文章一覧画面、妖怪図鑑画面
  - **音響想定**: 篠笛や鉦（かね）、和太鼓を緩やかに用いつつ、低音のドローン（持続音）や怪しげな揺らぎ（ビブラート）を混ぜた、どこか冷たく妖しげな和風お囃子調の曲。プレイヤーに静かな恐怖と緊張感を与えます。
- **戦闘用和風BGM（戦闘時）**:
  - **対象画面**: タイピング（戦闘）画面（ステージモード / 時間制限モード共通）
  - **音響想定**: 太鼓、不協和な篠笛、耳障りな鉦、狂ったようにかき鳴らされる三味線などが鳴り響く、アップテンポで活気がありつつも、極めて不気味で緊迫感に満ちたダークお囃子調の戦闘曲。

---

## 2. 効果音（SE）
プレイヤーのアクションや戦闘イベントに応じた、不気味で質感のある効果音（SE）を定義します。

- **ボタン押下音**:
  - **対象動作**: 各メニューやダイアログのボタンをクリック・選択した瞬間。
  - **音響想定**: 暗く少しエコー（残響）をともなう、カチッとした、または背筋が少し冷っとするようなシステム決定音。
- **タイピング音（キー入力成功時）**:
  - **対象動作**: タイピング中、正しいキーを入力した瞬間（一打ごと）。
  - **音響想定**: キーボードを叩くような、小気味よい「カタカタ」という乾いた打鍵音。少し古風なタイプライターのような、あるいは不気味な部屋に響く乾いた音を想定。
- **タイピングミス音**:
  - **対象動作**: タイピング中、誤ったキー（タイピングミス）を入力した瞬間。
  - **音響想定**: 「ぶっ」という、明らかに間違ったことをプレイヤーが即座に認識できる電子的なブザーや鈍い不協和音。音量を上げ、エッジの効いたより硬質なデジタル不協和音とします。
- **妖怪への攻撃音（お祓い成功時）**:
  - **対象動作**: 1文すべてのタイピングが完了し、妖怪にダメージを与える瞬間。
  - **音響想定**: 「シャン」という澄んだ神聖な鈴（あるいは神楽鈴・錫杖）の金属音。音量を高め、鳴り始めのアタックを硬く鋭い金属音に強化します。
- **ダメージ被弾音（反撃被弾時）**:
  - **対象動作**: 妖怪が反撃し、プレイヤーのHPが減少する瞬間。
  - **音響想定**: 「ドスっ」という、体に重い一撃を喰らったような鈍く湿った打撃衝撃音。音量を高め、重低音のアタックをより硬く重く強化します。
- **完全防御音（妖怪の反撃を完全に防いだ時）**:
  - **対象動作**: 妖怪が反撃してきた際、お祓いや防御判定などによってダメージを完全に防いだ（0にした）瞬間.
  - **音響想定**: 「キンっ」という、金属の盾や刀で攻撃を鋭く弾き返したような、高音で硬質な金属摩擦・打撃音。極めて高音の鋭いアタック音で、音量もしっかりと強調されます。

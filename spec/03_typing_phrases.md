# 妖怪タイピングゲーム 仕様書：03_タイピング文言設計

## 1. タイピング出題テーマ・カテゴリ
出題されるタイピング文言（1分ごとに別の文言がランダム表示、攻防サイクルごとに出題）は、以下の3つの主要テーマに基づいています。

1. **妖怪の名前**:
   - 登場する妖怪や、その他有名な日本の伝統的な妖怪名。
2. **悪霊退散・浄化に関する文言**:
   - お祓い、呪文、儀式、退散、結界、精神統一などにまつわる和風のフレーズ。
3. **神社・お寺に関する文**:
   - 建造物（鳥居、本堂）、参拝方法、仏教・神道の用語、歴史ある寺社に関する説明文や語句。

---

## 2. 難易度（ステージ毎の構成）
ステージ番号が上がるに従って、表示される一文の「文字数」や「ローマ字入力の複雑さ（拗音・長音・文字種など）」、および「文章の意味としての難解さ」が上昇します。

### 段階別 難易度設計

#### 【初級】ステージ 1 〜 3 （短文・平易な文言）
- **文字数（ひらがな基準）**: 4文字 〜 10文字
- **特徴**: 拗音（「きゃ」「しゅ」など）が少なく、基本のキー配置でタイピングできる身近な単語や極短文。
- **フレーズ例**:
  - `ねこまた` (NEKOMATA) - 妖怪
  - `おはらいをする` (OHARAIWOSURU) - 浄化
  - `あかいとりい` (AKAITORII) - 神社

#### 【中級】ステージ 4 〜 8 （日常フレーズ・中規模文）
- **文字数（ひらがな基準）**: 11文字 〜 20文字
- **特徴**: 漢字（ルビ表示あり）が混ざった一般的な文章。タイピングにある程度の速度と手の動きが求められる。
- **フレーズ例**:
  - `ろくろくびがくびをのばす` (ROKUROKUBIGAKUBIWONOBASU) - 妖怪
  - `あくりょうたいさんのふだ` (AKURYOUTAISANNOFUDA) - 浄化
  - `てらでおみくじをひく` (TERADEOMIKUJIWOHIKU) - 神社・お寺

#### 【上級】ステージ 9 〜 13 （長文・難読・仏教神職用語）
- **文字数（ひらがな基準）**: 21文字 〜 35文字以上
- **特徴**: 長文であり、専門用語や古風な表現、読み方に惑わされやすい拗音や促音が多く含まれる。
- **フレーズ例**:
  - `きゅうびのきつねがまやかしのほのおをはなつ` (KYUUBINOKITSUNEGAMAYAKASHINOHONOOWOHANATSU) - 妖怪
  - `きゅうきょくりょうほうをもってけっかいをはる` (KYUUKYOKURYOUHOUMOTTEKEKKAIWOHARU) - 浄化
  - `ごじゅうのとうのしゃりねんじゅのけはい` (GOJUUNOTOUNOSHARIWONESSHU...等) - 神社・お寺

## 3. システム表現要件・キー入力仕様
- **表示形態**: 
  - 画面には「漢字かな交じり文（メイン）」と「ひらがな（サブ）」および「リアルタイムのローマ字ガイド」を併記することで、誰でも迷わず入力できる支援を行います。
- **文言のランダム抽選（ループ仕様）**:
  - 各ステージ開始時、および攻防サイクルが1回終了して入力がリセットされる際に、そのステージに用意された文章リストから1行がランダムに抽出され、画面下部にセットされます。
  - **繰り返し出題**: 妖怪の体力が0（撃破）になるまで、用意された文章は途切れることなく繰り返し（ループ）出題され続けます。一度使用された文章も再出題されます。

### 3.1 複数ローマ字入力パターンの許容ルール
一般的な日本語タイピングシステムに基づき、同一のひらがな（音）に対して複数のキー入力パターンが存在する場合、**すべてのパターンでの入力を可能（許容）** とします。

#### 主要な許容パターンの例
- **し**: `shi` / `si`
- **ち**: `chi` / `ti`
- **つ**: `tsu` / `tu`
- **じ**: `ji` / `zi`
- **ぢ**: `di` / `zi`
- **しゃ / しゅ / しょ**: `sha` / `sya` , `shu` / `syu` , `sho` / `syo`
- **ちゃ / ちゅ / ちょ**: `cha` / `tya` , `chu` / `tyu` , `cho` / `tyo`
- **じゃ / じゅ / じょ**: `ja` / `zya` , `ju` / `zyu` , `jo` / `zyo`
- **ふ**: `fu` / `hu`
- **ん**: `nn` / `xn` (単独、または文末等で次のキーに影響しない場合)
- **か / く / こ / せ / そ 等の拗音を伴う各種入力、および訓令式・ヘボン式の混在**:
  - プレイヤーが最初に入力し始めた方式（ヘボン式か訓令式か等）を動的に検知し、打鍵途中で矛盾しない限りすべての正規表記パターンを正しく受け付けます。
  - ローマ字ガイドは、プレイヤーが入力したキーに応じて、対応する入力パターンへとリアルタイムに動的書き換え（最適化）を行います。

# 妖怪タイピングゲーム 仕様書：05_ランダムモード＆回顧録

## 1. 解放条件
- **条件**: 全13ステージ（牛鬼まで）をすべてクリアした後に、メインメニューまたはステージ選択画面において「ランダムモード」が解放されます。

---

## 2. ランダムモードの全体構成
ランダムモードを選択すると、プレイヤーは以下の2つの機能を選ぶことができます。
1. **タイピングモード**: これまで登場したすべての妖怪を相手にする3分間のサバイバルタイピングゲーム。
2. **回顧録（ログ）**: タイピングモードで記録された過去のスコアや各種統計データを閲覧する記録簿。

---

## 3. タイピングモード詳細仕様

### 3.1 出現妖怪・進行システム
- **出現妖怪**: ステージ1〜13で登場したすべての妖怪（猫又〜牛鬼）が、ステージ順ではなく完全にランダムで次々と出現します。
- **基本ルール**: 
  - 敵のHP・攻撃力などは、ステージ制（[キャラクター＆ステージデータ](/spec/02_character_data.md)）で定義された各妖怪のステータスに準拠します。
  - 基本システム（タイピング評価・与ダメージ・被ダメージ・ノーミスガード）は通常ステージと同様です。
  - **撃破時挙動**: 妖怪を倒すと、次の妖怪が即座にランダムで出現します。

### 3.2 終了・リザルト条件
以下のいずれかの条件を満たすと戦闘が終了し、タイピング結果画面へと移行します。
1. **制限時間の経過**: 計 **3分間** が経過した時点。
2. **プレイヤーの敗北**: プレイヤーの現在HPが **0** になった時点。
   - **相打ち・優先処理**: 妖怪を倒した（敵HPが0）と同時にプレイヤーのHPも0になった場合、または一連の処理の中でプレイヤーのHPが0になった時点で、妖怪撃破よりもプレイヤー敗北が最優先され、HP0のまま次の妖怪へ進むことなく即座にタイピングモードが終了（リザルト画面へ遷移）します。

---

## 4. タイピング結果（リザルト）と回顧録の記録項目

### 4.1 リザルト表示項目
戦闘終了時に以下の7つの情報が画面に表示されます。

1. **入力速度 (CPM)**: 1分間に何文字入力できたか（字/分） ※3分間全体の平均値
2. **精度 (%)**: 総キー入力数に対する、正しいキー入力の割合（％）
3. **総文字入力数 (字)**: 制限時間中にプレイヤーが入力した正しい総文字数
4. **入力ミス数 (字)**: 制限時間中に誤って入力した総ミス数
5. **最頻ミスキー**: プレイ中に最も多くミス入力した特定のキー（例：「K」や「S」など）
6. **総合スコア**: 2〜3桁（最大999）の数値として計算される総合指標
7. **日付**: プレイ完了時の日時（年/月/日 時:分）

### 4.2 総合スコア計算式（2〜3桁表示）
スコアはプレイヤーの「入力速度（平均CPM）」と「入力精度（%）」をベースに、以下の計算式で2桁〜3桁の実数値として算出します（小数点以下は四捨五入）。

$$\text{総合スコア} = \text{平均CPM} \times \left( \frac{\text{精度 (\%)}}{100} \right)$$

*※例：平均CPMが 350、精度が 96% の場合*
$$350 \times 0.96 = 336 \quad (\text{スコア: } 336)$$
*※例：平均CPMが 120、精度が 90% の場合*
$$120 \times 0.90 = 108 \quad (\text{スコア: } 108)$$
*※例：平均CPMが 80、精度が 85% の場合*
$$80 \times 0.85 = 68 \quad (\text{スコア: } 68)$$

## 5. 回顧録（記録システム）

### 5.1 記録の保存・並び替えルール
- **最大保存件数**: 直近 **25件** まで。
- **保存の挙動**: 新しくスコアが記録されると、過去の古い記録は残したまま順次追加されます。ただし、保存件数が25件を超えた場合は、最も古い日時（日付）のデータから自動的に削除されます。
- **並び替え（ソート）**: 回顧録の一覧画面では、**「日付が新しい順（降順）」** で常に表示されます。

### 5.2 画面要件（回顧録ビュー）
- 「回顧録」を選択すると、これまで記録された過去のプレイデータ一覧をリスト形式で閲覧できます。
- リストでは、日付、総合スコア、そして詳細データ（CPM、精度、総入力数、ミス数、最頻ミスキー）が確認できます。
